# AgentLabX Web

React + TypeScript + Vite + Ant Design frontend for AgentLabX.

## Quickstart

```bash
npm install
npm run dev
```

The dev server runs at http://localhost:5173 and proxies:

- `/api/*` -> `http://localhost:8000` (FastAPI backend)
- `/ws/*`  -> `ws://localhost:8000` (WebSocket)

The backend must be running at `localhost:8000` for API calls to succeed.

## Scripts

- `npm run dev` — start Vite dev server
- `npm run build` — type-check + production build to `dist/`
- `npm run preview` — preview the production build
- `npm run test` — run Vitest
- `npm run test:ui` — run Vitest with UI
- `npm run typecheck` — `tsc --noEmit`
- `npm run lint` — ESLint over `src/`
- `npm run codegen` — regenerate `src/api/generated.ts` from the backend's `/openapi.json` (added in Task 2)

## API types

`src/api/generated.ts` is generated by `npm run codegen` and is **committed to git** (not gitignored), so backend API changes surface clearly in PR diffs.
